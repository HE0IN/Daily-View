# Daily View

개발자와 검토자(QA/기획자)가 개발 요청 사항을 주고받기 위한 로컬 호스팅 웹앱입니다.

기존에는 PPT에 스크린샷을 붙여넣고 텍스트로 코멘트를 다는 방식으로 관리했지만,
- 항목 정리가 어렵고
- 누가 뭘 처리 중인지 한눈에 파악하기 힘들고
- 긴급도/상태 필터링이 안 되는

문제가 있어, 이를 웹앱으로 전환합니다.

## 구성

- **프레임워크**: Python 3.12.7 + Streamlit
- **저장소**: 로컬 폴더 (DB 없음 — JSON + 이미지 파일)
- **호스팅**: 공용 PC에서 `0.0.0.0:8501`로 띄워 사내 네트워크에서 접속
- **사용자 식별**: 로그인 없이 사이드바에서 이름 + 역할(검토자/개발자) 선택

## 기능 요약

- 검토자: 스크린샷 + 설명 + 긴급도(상/중/하)로 새 요청 등록
- 개발자: 요청을 보고 상태 변경(확인중/완료/API사확인요청) 및 코멘트 작성
- 검토자: 개발자 답변 후 실제로 동작 확인 → "검토완료" 처리
- 대시보드에서 긴급도/상태별 카운트, 내 담당 큐 즉시 확인
- 모든 변경 이력이 audit log로 보존됨

## 빠른 시작 (구현 후)

```bash
py -3.12 -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py --server.address 0.0.0.0 --server.port 8501
```

상세 절차는 [docs/05_setup.md](docs/05_setup.md) 참고.
